# LLM-Powered DevOps Assistant for CI/CD Automation

## Description

An AI-powered full-stack web application that integrates Jenkins, Django, PostgreSQL (Docker), and Large Language Models (LLMs) to automate CI/CD build log analysis, identify root causes, and generate intelligent troubleshooting suggestions.

## Tech Stack

- Python
- Django
- PostgreSQL (Docker)
- Docker Desktop
- Jenkins
- Git
- GitHub
- HTML
- CSS
- JavaScript
- Bootstrap
- REST APIs
- OpenAI / Gemini API
